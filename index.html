Dokumentasi

Identity

  1. Name

    ✔️ Already Implemented

  2. Email

    ✔️ Already Implemented

Information

  1. Report

    ✔️ Already Implemented

  2. Severity

    Data Type: String

    Pada field Severity, user akan memilih opsi Severity Level berdasarkan nilai/value drop down list yang ada (Perlu diperbaiki sesegera mungkin, Parah, Sedang, Kecil, dan Saran).

    Salah satu opsinya menjadi placeholder dari input field Severity Level

    Label Severity Level berada di sebelah kiri drop downnya (satu line)

    Kirim valuenya ke Backend

  3. Platform and Device Environment

    Data Type: String, except RAM. RAM is number/integer

    Pada field Platform and Device Environment, ada 3 field bertipe Text yaitu Phone Brand, RAM, dan Android Version

    Kirim valuenya ke Backend

  4. Attach files, videos or images (Optional, tidak wajib diisi oleh user)

    Data Type: [An image, file, and video is typically a binary format file]

    Acceptable Format: .txt/.pdf/.JPG/.png/.mp4/.mov/

    Pada field Attach files, videos or images, user akan mengupload sebuah file (txt, pdf, video, atau gambar)

    User akan dibatasi oleh Size Limit sebanyak 1 MB dan Maximum File sebanyak satu buah

    Kirim valuenya ke Backend
